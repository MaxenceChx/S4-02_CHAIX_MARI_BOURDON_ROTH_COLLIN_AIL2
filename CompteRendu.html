Schéma de l'architecture

Voici un schéma complet de l'architecture de notre projet, avec les technologies employées et les échanges :

Ce qui a été fait

Résumé des tâches effectuées :

  • Affichage d'une map de Nancy avec Leaflet, Changement du fond de la carte avec MapTiler qui est une technologie intégré a Leaflet, Utilisation du plugin IconMaterial récupérer sur github pour afficher les icons d'une manière plus jolie
  • Récupération des données pour les bornes de vélib a l'aide de l'API Fetch et affichage de celle-ci sur la carte
  • Affichage de tout les arrêts de bus sur la carte a l'aide d'une base de donnée supabase déjà créer par Maxence et réutilisé pour le projet
  • Création de la base de données pour les réstaurants
  • Création de la barre de navigation avec un onglet "carte","compte rendu" et un onglet "Ajouter un restaurant" pour afficher une modal permettant de rentrer les informations nécessaire a l'ajout d'un restaurant en base de donnée
  • Ajout d'un module de filtre intégré a Leaflet pour choisir les éléments a afficher sur la map
  • Création d'un serveur java utilisant RMI et HttpServer permettant de jouer un rôle de proxy entre les différentes données bloquées et le site web (responsable ?)
  • Création d'un client à l'aide de HttpClient pour récupérer les données des incidents de la route a nancy
  • De la même manière on créer un client pour les établissements supérieur
  • On créer un dernier client qui gère les réstaurants en se connectant a la base de données
  • Ajout d'un icône météo a la barre de naviguation pour afficher les informations météoroligique a Nancy du jour actuel
  • Création d'un bouton pour afficher le détail des status des points sur la carte : '✅' si le service est opérationnel, '❌' sinon

Informations nécessaires à l'utilisation du projet

A faire avant de pouvoir lancer le projet :

  • Etape 1 : Executer le scripts ayant pour chemin BD/script_base_resto.sql depuis la racine du projet pour créer la base de données des restaurants.
    Executer la classe ayant pour chemin java/Restaurant/DBConnection depuis la racine du porojet pour insérer les restaurants dans la table
  • Etape 2 : Ajouter la librairie JSON : Project Structure -> Libraries -> Boutton "+" -> Choisir "From Maven" et chercher "org.json:json:20240303" -> Boutton "OK"
  • Etape 3 : Services a lancer
  • Etape 4 : Afficher la carte
